Installation and setup on Window

Requiments

*You must be running Windows 10 version 2004 and higher (Build 19041 and higher) or Windows 11

Setting it up WSL

  1. Run Windows PowerShell in Administrator
  2. WSL is the Windows Subsystem for Linux and it will allow us to run command line commands in Windows.
  3. Copy and paste code below into PowerShell and hit the enter key to run it.
    4. Copy Text
    wsl --install
  4. Restart PC once completed
  5. Upon restart you will be prompted to setup an ubuntu username and password
    and then you will have successfully installed WSL. Make sure to keep username and passord safe somewhere.
  6. Confirm you have everything set up properly
    7. Copy Text
    wsl --list --verbose
  7. If all went well you should see this(make sure you are on version 2!)

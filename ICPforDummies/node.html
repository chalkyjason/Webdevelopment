Installing Node

Homebrew

  1. Open Up Ubuntu
  2. Copy and past command to install homebrew
    3. Copy Text
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  3. You will be asked for password you set early to install
  4. The installer will tell you how to add brew to the path
  5. Next you will install Homebrew's dependencies
    6. Copy Text
    sudo apt-get install build-essential
  6. To check to make sure it installed correctly enter this!
    7. Copy Text
    brew --version
  7. Now intstall the NODE
    8. Copy Text
    brew install node@16
  8. Now check the node version
    9. Copy Text
    node --version

Create your first DApp

Time to test to make sure everything works, because I am a dummy and might have screwed it up

    Create the Default Hello World DApp

  1. open up Ubuntu to make a new project by creating a new folder with this command
    2. Copy Text
    mkdir testProject
  2. change the directpry into the new project you just created
    3. Copy Text
    cd testProject
  3. inside this folder we are going to create a prebuilt project for you using this command
    4. Copy Text
    dfx new hello
  4. To see where this new project is installed you can use this command
    5. Copy Text
    explorer.exe .
  5. Now you will want to open up VScode and click the bottome left like the image shows
  6. select NEW WSL Window
  7. On the left you will go into the extension tabe and instakk WSL:Ubuntu if you have not done so in the past

    8. Time to Deploy the DApp to local Network

  8. In VSCoder go to the terminal tab and select New Terminal
  9. In that terminal you are go to post this commad to start DFX
    10. Copy Text
    dfx start
  10. Now you will need to open another terminal by click the plus button, or going to the tab up top again
  11. Once the new terminal is open you need to get inside the hello project by running this command
    12. Copy Text
    cd testProject/hello
  12. Now you can deploy you first project running this command in the terminal
    13. Copy Text
    dfx deploy
  13. Once that deploys run one last command
    14. Copy Text
    npm start
  14. Congrats if all was succesfull, you launched your first DApp on your local network, oh you want to see it I guess.
    15. http://localhost:8080/
